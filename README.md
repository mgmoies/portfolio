# 🛡️ Tapyothin Vissanuyothin — Personal Portfolio

> Cybersecurity Student (Year 4) · DevSecOps Intern @ Seagate Technology

A minimal, clean, Apple-inspired personal portfolio built with **React + Vite**.

## ✨ Features

- 🎨 Minimal & Clean design (Apple-inspired)
- 📱 Fully responsive (mobile-first)
- 🔐 Cybersecurity & DevSecOps focused content
- 🚀 Smooth scroll progress indicator
- 💌 Contact form via mailto
- ⚡ Fast — Vite build

## 📄 Sections

| Section | Description |
|---|---|
| Hero | Introduction, status badge, stats |
| About | Background, highlights |
| Skills | Categorized tech skills |
| Experience | Career timeline |
| Testimonials | Feedback from mentors/clients |
| Contact | Form + contact info |

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite 8
- **Styling**: Vanilla CSS (design tokens)
- **Fonts**: Inter + JetBrains Mono (Google Fonts)
- **Deployment**: GitHub Pages (or Vercel)

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 🌐 Deployment (GitHub Pages)

```bash
npm run build
# Then push the dist/ folder, or use github-pages action
```
